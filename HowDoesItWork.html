The Website

  • Each question is scored between 0 and 100 depending on how far the bar is dragged
  • Questions fall into categories corresponding to the traits shown on the results page
  • Each question will contribute either positively or negatively to its corresponding trait
  • The results from the questions are tallied and passed along to the results page
  • In total, each of the 5 traits will have a score between 0 and 500
  • If the score for any given trait is equal to or above 250, the positive of that trait is present, otherwise the negative is
  • Depending on how far the result is from 250, a higher or lower percentage score is given
  • This website is made using raw HTML, CSS, and JavaScript
  • The code can be found here.
  • A few notable features are the animated buttons and the custom progress bars for showing trait percentages with tooltips
